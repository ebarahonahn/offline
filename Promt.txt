Desarrolla un sistema web institucional moderno para el control de tiempo laboral, actividades e inactividad de empleados en modalidad de teletrabajo, enfocado en transparencia, productividad y supervisión en tiempo real.

🧩 Arquitectura Tecnológica
Frontend: Angular (última versión estable) + Tailwind CSS (diseño moderno, limpio y responsive)
Backend: Node.js con Express
Base de Datos: MySQL
Comunicación: API RESTful
Autenticación: JWT (JSON Web Tokens)
Tiempo real (opcional): WebSockets (Socket.io)
🎯 Objetivo del Sistema

Permitir a las instituciones:

Monitorear la jornada laboral de empleados remotos
Registrar actividades realizadas
Detectar tiempos de inactividad
Generar reportes de productividad
Supervisar cumplimiento de horarios
👥 Roles del Sistema
Administrador
Gestión de usuarios
Configuración de horarios laborales
Visualización global de reportes
Supervisor
Monitoreo de empleados asignados
Validación de actividades
Empleado
Registro de inicio/fin de jornada
Registro de actividades
Visualización de su propio desempeño

1. 🔐 Autenticación y Seguridad

Roles y permisos

Hora de finalización

"Pausar"
"Reanudar"

4. 💤 Detección de Inactividad

Registro automático de tiempos muertos
Alertas visuales al usuario
5. 📊 Panel de Control (Dashboard)
Gráficas de:

Actividades realizadas
Filtros por:
Fecha
Usuario
Departamento
6. 📈 Reportes
Reportes exportables (PDF / Excel)

Horas trabajadas vs inactividad

7. 🛠️ Configuración
Parámetros del sistema:
Tiempo de inactividad
Horarios laborales
Días laborales
Gestión de catálogos:

Departamentos
🧮 Base de Datos (Modelo Inicial)

Tablas principales:

usuarios (id, nombre, email, password, rol_id, estado)
roles (id, nombre)
jornadas (id, usuario_id, fecha, hora_inicio, hora_fin)
pausas (id, jornada_id, hora_inicio, hora_fin)
actividades (id, jornada_id, nombre, descripcion, tiempo)
inactividad (id, usuario_id, fecha, tiempo_inactivo)
configuraciones (clave, valor)
